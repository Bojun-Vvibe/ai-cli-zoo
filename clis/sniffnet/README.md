# sniffnet

- **Repo:** https://github.com/GyulyVGC/sniffnet
- **Latest version:** v1.5.0
- **License:** Apache-2.0 OR MIT (`LICENSE-APACHE`, `LICENSE-MIT`)
- **Category:** dev-tools / networking

A cross-platform network traffic monitor with a friendly graphical TUI that
groups packets by protocol, host, port, and country, plots throughput over
time, and lets you set custom notifications when traffic patterns cross a
threshold. Reach for sniffnet when `bandwhich` is too read-only and
`wireshark` is too much — it sits in the middle, good for spotting which
app is chatty or which remote host is eating bandwidth. Runs on Linux,
macOS, and Windows from a single Rust binary.
